# EarlyCompat

Eraly Compat 为移植的铜灯纹饰陶罐方块添加本地协议兼容，同时它还修复了一些其它可能导致协议出错的bug，确保 pca-masa 和 carpet-litematica 等模组能正常识别和交互。
Eraly Compat adds local protocol compatibility for ported copper bulb and decorated pot blocks, while also fixing some other bugs that could cause protocol errors, ensuring proper recognition and interaction with mods like pca-masa and carpet-litematica.
